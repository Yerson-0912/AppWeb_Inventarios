Flujo recomendado

Este es el orden más rápido para evitar errores y generar tus reportes en un solo intento.

  • Carga el archivo Excel de inventario actualizado.
  • Selecciona empresa y define la cantidad mínima en PRINCIPAL.
  • Pulsa Leer vendedores y marca solo las bodegas que vas a procesar.
  • Activa Nueva colección si quieres resaltar referencias recientes.
  • Genera y descarga el ZIP con todos los PDFs listos para compartir.
Tip: si cambias el archivo Excel, vuelve a pulsar Leer vendedores para refrescar la tabla antes de generar.
